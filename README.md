# 💻 Portfólio Pessoal - Gabriel Vieira

Este projeto foi desenvolvido como atividade prática da disciplina de Fundamentos da Programação Web, com o objetivo de aplicar conceitos de HTML5, CSS3 e JavaScript na criação de um portfólio online.

---

## 🚀 Acesse o projeto

🔗 Site publicado: https://gabrielvieira-dev.github.io/projeto-portfolio-uninter
🔗 Repositório: https://github.com/gabrielvieira-dev/projeto-portfolio-uninter

---

## 📌 Sobre o projeto

O portfólio foi desenvolvido com foco em apresentar minhas informações pessoais, formação acadêmica, projetos realizados e um formulário de contato funcional.

O site possui navegação entre páginas e foi construído seguindo boas práticas de desenvolvimento web, com código organizado e uso de tags semânticas.

---

## 🧩 Estrutura do site

O projeto é composto por 4 páginas principais:

- **Sobre mim** → Apresentação pessoal e objetivos profissionais  
- **Formação** → Cursos, graduação e nível de idioma  
- **Portfólio** → Projetos desenvolvidos com links para acesso  
- **Contato** → Formulário com validação em JavaScript  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ Funcionalidades

- Navegação entre páginas  
- Layout responsivo  
- Estilização moderna com CSS  
- Validação de formulário com JavaScript  
- Simulação de envio de mensagem
